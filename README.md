🍕 SQL Pizza Sales Analysis Project
📊 Project Overview
This project focuses on analyzing pizza sales data using SQL. It covers a range of queries from basic data retrieval to advanced revenue analysis. The goal is to derive meaningful business insights, such as revenue trends, order patterns, and top-selling pizzas.

⚡ Key Features
Calculate total orders, revenue, and most popular pizzas
Perform category-wise and time-based sales analysis
Identify revenue-driving products and track cumulative growth
